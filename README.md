# Wanderlust 🌍✈️

A full-stack travel exploration app built with **Node.js, Express, MongoDB, and Cloudinary**.  
Users can discover destinations, upload images, and share experiences securely.

## Features
- 🔐 Secure authentication & session management
- 🗺️ Destination listings with CRUD operations
- 📸 Image uploads via Cloudinary
- 🛡️ Robust schema validation & error handling
- 🚀 Deployment-ready with MongoDB Atlas & Render

## Tech Stack
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Frontend: React
- Image Hosting: Cloudinary
